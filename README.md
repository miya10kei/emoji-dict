emoji-dict
---
Generate emoji dictionary from [GitHub Emojis API](https://docs.github.com/en/rest/reference/emojis).  
dist: https://github.com/miya10kei/emoji-dict/blob/main/dist/emoji.list

## Requirement
- Java 11

## How to
### Build
```bash
./gradlew clean buld
```

### Execute
```
./gradlew run
```
